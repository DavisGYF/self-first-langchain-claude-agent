# 📸 OCR 图片文字识别功能

## 🎯 功能概述

这个OCR功能实现了真正的多模态融合，让你能够：

1. **📷 上传图片** - 支持PNG、JPG、JPEG、WebP格式
2. **🔍 自动识别** - 使用Tesseract.js识别图片中的文字
3. **💬 文本融合** - 将识别的文字与你的描述结合
4. **🤖 AI理解** - AI同时理解图片内容和你的文字描述

## 🔧 技术实现

### 前端组件 (ImageOCR.tsx)
- **拖拽上传** - 支持拖拽或点击上传图片
- **实时预览** - 显示上传的图片
- **进度指示** - 显示OCR处理状态
- **结果展示** - 显示识别的文字和置信度
- **一键复制** - 快速复制识别结果

### 后端API (route.ts)
- **Tesseract.js集成** - 强大的OCR引擎
- **多语言支持** - 支持中文、英文、日文、韩文
- **临时文件管理** - 安全的文件处理
- **错误处理** - 完善的异常捕获

## 🚀 使用步骤

### 第一步：上传图片
1. 在聊天输入框旁边点击 **📷 图片图标**
2. 选择或拖拽包含文字的图片
3. 系统自动开始OCR识别

### 第二步：查看结果
- 识别完成后，文字会自动添加到输入框
- 你可以看到识别的置信度和处理时间
- 支持查看详细的分块识别信息

### 第三步：多模态对话
- 在识别的文字基础上添加你的描述
- 发送消息，AI会同时理解图片内容和你的描述
- 获得更准确的回答
